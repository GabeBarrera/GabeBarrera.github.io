Setup & Requirements
1
Install Sherlock on Kali:
sudo apt install sherlock   or   pip install sherlock-project
2
Start the local API server (stdlib Python, no extra installs):
python3 sherlock-server.py   — listens on 127.0.0.1:7474
3
Open this file in your browser, enter a username, and click HUNT. Results stream in real-time from Sherlock's output.
4
The server only accepts usernames matching [a-zA-Z0-9._\-]{1–64} and binds to localhost only — it never exposes Sherlock to the wider network.
